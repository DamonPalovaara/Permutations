- First you must install Rust, follow the instructions here:
https://www.rust-lang.org/tools/install

- Next download the repository onto your computer.

    - Click the green Code button
    - Click the Download ZIP button on the menu
    - Download it somewhere
    - Make sure to extract the ZIP

- Once downloaded you can right click inside of the main repository folder and select "open up in terminal".

- Now all you need to do is execute the command ```cargo run```
